Queue & Stack

Posted on 2020-11-01 In Data Structure Disqus:

1. Basic Concepts

Queue

  1. Wait in line, FIFO
  2. Usages: Breadth-First Search related problems

Stack

  1. LIFO: Last in first out 竹筒/羽毛球筒
  2. All operations can only be done to top element.
    push(), pop(), peek()
  3. Usage: 从左至右linear scan一个array/string时, 如果要不断回头看左边最新的元素时, 往往用到stack
    a. String repeatedly deduplication (cabba -> caa -> c);
    b. Reverse Polish Notation 逆波兰表达式的计算;
    c. Histogram中找最大长方形.

Key points:
a. 将Stack1所有元素全部move到Stack2, 元素在Stack2中的顺序完全reverse
b. 将Stack1所有元素move到Stack2, 然后元素全部(或部分)move回Stack1, 则回到原来Stack1的元素的顺序不变.

Amortized time VS average time
Amortized time: for any specific input, we amortize all operations within this input
Average time: the average of all possible inputs in the input space (e.g., quickSort)
