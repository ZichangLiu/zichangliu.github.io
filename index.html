Queue & Stack

Posted on 2020-11-01 In Data Structure Disqus:

1. Basic Concepts

Queue

  1. Wait in line, FIFO
  2. Usages: Breadth-First Search related problems

Stack

  1. LIFO: Last in first out 竹筒/羽毛球筒
  2. All operations can only be done to top element.
    push(), pop(), peek()
  3. Usage: 从左至右linear scan一个array/string时, 如果要不断回头看左边最新的元素时, 往往用到stack
    a. String repeatedly deduplication (cabba -> caa -> c);
    b. Reverse Polish Notation 逆波兰表达式的计算;
    c. Histogram中找最大长方形.

Linked List

1. Basic Concepts

N1 -> N2 -> N3 -> N4 -> … -> N1000 -> NULL

链表问题都是inplace的, 所有结点在内存中都是原地不动的, 变化的只有结点里存的下一结点的地址

Key points:

  1. When you want to dereference a ListNode, make sure it is not a NULL pointer
    //ListNode p = new ListNode(10);
    p.value
    p.next.value //“.”is an action that is called de-reference
  2. Never ever lose the control of the head pointer of the linkedList, or you cannot get access to other nodes anymore.
