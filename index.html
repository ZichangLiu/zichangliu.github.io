Dynamic Programming

Posted on 2020-11-08 In Algorithm Disqus:

Methodology

Core Idea

DP的核心思想类似于数学归纳法:

  1. 把一个大问题(size == n)的解决方案用比他小的问题们来解决, 也就是思考从问题size = n-1增加到size = n 的时候, 如何用小问题的solution构建大问题的solution
  2. 就是利用历史记录，来避免我们的重复计算。而这些历史记录，我们得需要一些变量来保存，一般是用一维数组或者二维数组来保存。
  3. 与recursion的关系:
    • Recursion从大到小来解决问题, 不记录任何sub-solution, 只要考虑
      • base case
      • recursive rule
    • DP从小到大来解决问题, 记录sub-solution
      • 由size(< n)的subsolution (s) -> size(n) solution
      • base case
      • induction rule

Introduction

In Java, String is class.

Popular implementation: char[] array

  1. ASCII representation of a letter (0 ~ 127 (2^7-1))
  2. Unicode: the latest version of Unicode contains a repertoire of more than 110,000 characters covering.
    • 对于ASCII里127号以下字符, Unicode维持原编码, 只是将长度由原来8位扩展为16位, 而其他语言的字符则采用全部重新统一编码.

Graph Search Algorithm I

Posted on 2020-11-05 Edited on 2020-11-08 In Algorithm Disqus:

BFS-I (Breadth-First Search)

expand(cur) -> generate(cur.left) and generate(cur.right)

BFS-I的操作过程

Data structure:
Maintain a FIFO queue, put all traversed nodes that haven’t been expended
Steps

  1. Initial state (queue.add(root))
  2. Expand a cur node and release it from the queue
    Generate cur’s neighbor nodes: reach out to its neighboring nodes;
  3. Termination condition: do a loop until the queue is empty
  4. (Optional) deduplicate visited nodes (typically for graph not for free)
    1. e.g. each node is expanded only once
    2. e.g. each node is generated only once

time O(n), space O(n)
